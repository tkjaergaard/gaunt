Gaunt

Simple template engine for Wordpress based on Laravel 4 Blade

Download .zip Download .tar.gz View on GitHub

What is Gaunt?

Gaunt is the simplest template engine written for Wordpress. The engine is heavily inspired and based upon Laravel 4 Blade. It not nearly as heavy and feature-rich as dedicated template engines as Smarty and Twig. But it's help you simplify your theme pages quite a lot.

Installation

Gaunt is installable through Composer.

"require" : {
     "gaunt/gaunt":"1.0.*"
}
